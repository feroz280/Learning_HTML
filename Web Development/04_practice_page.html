HTML & CSS Basics Summary

    HTML Summary

    HTML (HyperText Markup Language) is used to define our pagec content, structure and meaning. You don't use it for styling purposes. Use CSS for that instead!

    • HTML uses"elements" to describe(annotate)content
    • HTML elements typically have an opening tag, content and then a closing tag
    • You can also have void (empty)elements like images
    • You can also configure elements with attributes
    • There's long list of available elements but you'll gain experience ober time,no worries.
